# dotnix

Flakes-based NixOS + Home Manager dotfiles converted from the previous Arch + stow layout.

## One-command setup

Run from the repository root:

```bash
./setup.sh
```

`setup.sh` will:

1. Detect current user, hostname, timezone, and architecture.
2. Generate `hosts/default/host-vars.nix`.
3. Copy `/etc/nixos/hardware-configuration.nix` to `hosts/default/hardware-configuration.nix`.
4. Apply the flake with `sudo nixos-rebuild switch --flake .#default`.

## Optional overrides

Set any of these environment variables before running `setup.sh`:

- `DOTNIX_USER`
- `DOTNIX_HOSTNAME`
- `DOTNIX_FULL_NAME`
- `DOTNIX_GIT_USERNAME`
- `DOTNIX_EMAIL`
- `DOTNIX_TIMEZONE`
- `DOTNIX_STATE_VERSION`
- `DOTNIX_ROOT`

## Layout

- `flake.nix`: flake entrypoint (`nixosConfigurations.default`, `homeConfigurations.<user@host>`)
- `hosts/default`: host-scoped values (generated by setup)
- `modules/nixos`: declarative system modules
- `modules/home`: Home Manager user module
- `configs`: source config files managed by Home Manager
- `scripts`: utility scripts installed into `~/.local/bin`
